## 常用命令

### 分支

#### 查看本地分支

```
git branch
```

#### 删除本地分支

```
git branch -d <branch_name>
```

#### 重命名本地分支

```
git branch -m <branch_old> <branch_new>
```


#### 查看本地及远程所有分支

```
git branch -a
```

#### 删除远程分支

```
git push origin --delete <branch_name>
```
